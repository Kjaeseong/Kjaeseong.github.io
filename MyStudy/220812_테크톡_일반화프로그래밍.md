# 일반화 프로그래밍
## 핵심 개념
- 일반화 프로그래밍이 무엇인지 이해한다.
- 일반화 메소드를 작성하고 사용하는 방법을 익힌다.
- 일반화 클래스를 작성하고 사용하는 방법을 익힌다.
- .NET에서 제공하는 일반화 컬렉션의 종류와 사용 방법을 익힌다.
---
## 일반화 프로그래밍이란
- 하나의 메서드(함수)를 다양한 타입에 사용하기 위한 프로그래밍 패러다임
  - 별도의 오버로딩 없이 다양한 데이터 타입에 호환되도록 할 수 있다
---

## 일반화 메서드
- 데이터 형식을 '일반화'한 메서드
- 구체적인 형식의 이름 대신 형식 매개변수(Type Parameter)가 들어간다.
- ex>
```cs
Qualifier returnType MethodName<TypeParameter> (parameterList)
//한정자   반환_형식   메서드이름  형식매개변수    매개변수 목록
```

#### 구현
```cs
//메서드
void swap(int a, int b)
{
  int temp = a;
  a = b;
  b = temp;
}
```
```cs
//일반화 메서드
static void swap<T>(T a, T b)
{
  T temp = a;
  a = b;
  b = temp;
}
```
- 위 코드는 입력받은 변수의 데이터를 교환하는 swap 메서드이다.
- 아래 코드 적용시 int뿐 아니라 다른 타입의 데이터도 삽입할 수 있다.

## 일반화 클래스
- 데이터 형식을 일반화한 클래스
- 형식 매개변수가 있는 것을 제외하면 보통의 클래스와 같다
- ex>
```cs
//    클래스이름  형식_매개변수
class ClassName<TypeParameter>
{
  // ...
}
```

#### 구현
```cs
namespace Generic
{
    class MyArray<T>
    {
        //배열 생성
        private T[] _array;

        //배열 크기 지정
        public void ArrayCreate(int size)
        {
            _array = new T[size];
        }

        //배열 데이터 삽입
        public void insert(int index, T Value)
        {
            if (index < _array.Length)
            {
                _array[index] = Value;
            }
        }

        public void Print()
        {
            for (int i = 0; i < _array.Length; i++)
            {
                Console.WriteLine(_array[i]);
            }
        }
    }

    class MainApp
    {
        static void Main(string[] args)
        {
            MyArray<string> Name_list = new MyArray<string>();
            Name_list.ArrayCreate(2);
            Name_list.insert(0, "kim");
            Name_list.insert(1, "park");
            Name_list.Print();

            MyArray<int> Num_list = new MyArray<int>();
            Num_list.ArrayCreate(3);
            Num_list.insert(0, 1);
            Num_list.insert(1, 2);
            Num_list.insert(2, 3);
            Num_list.Print();
        }
    }
}
```
- 위 코드는 다양한 데이터 타입을 다룰 수 있는 일반화 클래스 예시이다.
  - string, int Type의 배열 생성 후 출력 코드

## 형식 매개변수 제약???ㅅㅂ
- 특정 조건을 갖춘 형식에만 대응하는 매개변수가 필요할 때 사용
- 클래스 선언문 헤더에 where절을 추가한다.
- where절과 함께 사용할 수 있는 제약조건
  
| 제약                 | 설명                                                            |
|--------------------|---------------------------------------------------------------|
| where T : struct   | T는 값 형식이어야 한다                                                 |
| where T :  class   | T는 참조 형식이어야 한다.                                               |
| where T :  new()   | T는 반드시 매개변수가 없는 생성자가 있어야 한다.                                  |
| where T : 기반클래스 이름 | T는 명시한 기반 클래스의 파생 클래스여야 한다.                                   |
| where T : 인터페이스 이름 | T는 명시한 인터페이스를 반드시 구현해야 한다. 인터페이스 이름에는 여러 개의 인터페이스를 명시할 수도 있다. |
| where T : U        | T는 또 다른 형식 매개변수 U로부터 상속받은 클래스여야 한다.                          |

- 
- ex>
```cs
//클래스이름<형식_매개변수>      ParentClass로부터 파생된 클래스여야 할 것.
class ClassName<T>             where T : ParentClass
{
  // ...
}
```

## 일반화 컬렉션
- 일반화에 기반해서 만들어진 자료구조
- System.Collections.Generic 네임스페이스 사용
- 대표적인 네 가지 클래스
  - List<Ｔ>
  - Queue<Ｔ>
  - Stack<Ｔ>
  - Dictionary<TKey, TValue>
### List<Ｔ>
```cs
using System;
using System.Collections.Generic;

namespace UsingGenericList
{
    class MainApp
    {
        static void Main(string[] args)
        {
            List<string> list = new List<string>();
            list.Add("Kim");
            list.Add("Park");
            list.Add("Lee");

            for (int i = 0; i < list.Count; i++)
            {
                Console.WriteLine(list[i]);
            }
        }
    }
}
```

### Queue<Ｔ>
```cs
using System;
using System.Collections.Generic;

namespace UsingGenericQueue
{
    class MainApp
    {
        static void Main(string[] args)
        {
            Queue<string> queue = new Queue<string>();
            queue.Enqueue("Kim");
            queue.Enqueue("Park");
            queue.Enqueue("Lee");

            while (queue.Count > 0)
            {
                Console.WriteLine(queue.Dequeue());
            }
        }
    }
}
```

### Stack<Ｔ>
```cs
using System;
using System.Collections.Generic;

namespace UsingGenericStack
{
    class MainApp
    {
        static void Main(string[] args)
        {
            Stack<string> stack = new Stack<string>();

            stack.Push("Kim");
            stack.Push("Park");
            stack.Push("Lee");

            while (stack.Count > 0)
            {
                Console.WriteLine(stack.Pop());
            }
        }
    }
}
```

### Dictionary<TKey, TValue>
- 해시테이블 일반화 버전
- Key와 Value를 형식 매개변수로 가진다.
```cs
using System;
using System.Collections.Generic;

namespace UsingGenericDictionary
{
    class MainApp
    {
        static void Main(string[] args)
        {
            Dictionary<string, float> dic = new Dictionary<string, float>();
            dic["Kim"] = 80.3f;
            dic["Park"] = 90.7f;
            dic["Lee"] = 60.5f;

            Console.WriteLine(dic["Kim"]);
            Console.WriteLine(dic["Park"]);
            Console.WriteLine(dic["Lee"]);
        }
    }
}
```

## foreach를 사용할 수 있는 일반화 클래스
- System.Collections.Generic 네임스페이스 사용
- 해당 인터페이스 상속 시 foreach순회해도 형식 변환으로 인한 성능 저하가 없다
- IEnumerable<Ｔ> 인터페이스 메서드
  
| 메서드                            | 설명                                            |
|--------------------------------|-----------------------------------------------|
| IEnumerator GetEnumerator()    | IEnumerator형식의 객체 반환(IEnumerator로부터 상속받은 메서드) |
| IEnumerator<Ｔ> GetEnumerator() |  IEnumerator<Ｔ> 형식의 객체 반환                     |

- IEnumerable<Ｔ> 메서드와 프로퍼티
  
| 메서드                     | 설명                                                                        |
|-------------------------|---------------------------------------------------------------------------|
| boolean MoveNext()      | 다음 요소로 이동. 끝을 지난 경우에는 false, 이동이 성공한 경우 true                              |
| void Reset()            | 첫 번째 위치의 '앞'으로 이동. 첫번째가 0번이면 -1번으로 이동한다. 첫 번째로의 이동은 MoveNext()호출 후에 이뤄진다. |
| Object Current { get; } | 현재 요소 반환(IEnumerator로부터 상속받은 프로퍼티)                                        |
| T Current { get; }      | 현재 요소 반환                                                                  |
