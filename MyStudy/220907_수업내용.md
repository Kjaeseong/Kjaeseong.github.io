## 프로미스
- 자바스크립트, 노드에서는 주로 비동기를 접한다.
  - **동기** : 요청이 들어 왔을때 실행이 되고 반환값이 바로 나온다
  - **비동기** : 응답시간이 일정하지 않을 때.(주로 많이 쓰인다)
- ES2015부터 콜백 대신 프로미스를 기반으로 재구성된다.
  - 콜백 지옥 현상을 극복했다는 평가를 받는다.
- 실행은 바로 하되 결과값은 나중에 받는 객체
- 실행이 완료된 후 then이나 catch메서드를 통해 받는다.
- 

```javascript
const condition = true;

// new Promise로 프로미스 생성
const promise = new Promise((resolve, reject) => {
    // 콜백함수
    if(condition)
    {
        resolve('성공');
    }
    else
    {
        reject('실패');
    }
}
);
// 다른 코드가 들어갈 수 있음
promise
    .then((message) => {
        console.log(message);
    })
    .catch((error) => {
        console.error(error);
    })
    .finally(() => {
        console.log('무조건');
    });
```

프로미스 관련 내용 추가요망.


```javascript
const promise1 = Promise.resolve('성공1');
const promise2 = Promise.resolve('성공2');

Promise.all([promise1, promise2])
    .then((result) => {
        console.log(result);
    })
    .catch((error) => {
        console.error(error);
    })




```
