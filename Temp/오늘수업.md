# PunRPC
- RPC를 구현하는 속성.
  - [PunRPC]로 선언된 메서드는 다른 클라이언트에서 호출 가능

# RpcTarget
- All : 모든 사람에게(나 포함) 보내고 즉시 실행함. 나중에 조인한 사람은 실행하지 않고(주로 씀)
- Others : 모든 사람에게 보내긴 보내는데 나는 제외. 나중에 조인한 사람 실행 ㄴㄴ(주로 씀)
- MasterClient : 마스터 클라이언트만. RPC를 실행하기 전 얘가 끊길 수 있고 유실될 수 있다.
- AllBuffered : 모든 사람에게(나 포함). 새로운 사람도 실행(지연된 실행)
- OtherBuffered : 모든 사람에게 보내는데 나는 제외. 새로운 사람도 실행(지연된 실행)
- AllViaServer : 서버를 경유해 RPC 호출. 나 자신도 서버로부터 메세지를 받은 후에야 RPC 실행. 모든 클라이언트에 대해 RPC를 보내는게 동일하다.(조금 더 동일한 화면을 볼 수 있다)
