# 게임 서버에 대한 이해

## 멀티플레이 게임의 역사
- 로컬 멀티플레이어 게임 : 1대의 PC에서 2명 이상의 플레이어가 같이 플레이
- MUD : 멀티 유저 던전으로 텍스트 기반 멀티플레이 게임
- LAN : 이더넷 보급으로 근거리 통신망을 이용한 게임
  - ex> 스타크래프트 LAN게임
- 온라인 게임 : 랜 게임과 구현은 비슷하지만 레이턴시에 따른 부작용을 최소화하는 기법이 생겼다.
  - 레이턴시 마스킹 : 
  - **레이턴시** : 네트워크로 데이터를 전송하며 발생하는 시간지연(= ping)
