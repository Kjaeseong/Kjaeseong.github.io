# [문제 링크](https://www.acmicpc.net/problem/10718)

## 풀이
- 정수형 변수 A, B 선언 후 입력
- 출력단계에서 바로 더해서 출력한다.

```cpp
#include <iostream>
using namespace std;

int main()
{
	int a, b;
	cin >> a;
	cin >> b;

	cout << a + b;
}
```
