# [18108 : 1998년생인 내가 태국에서는 2541년생?! (링크)](https://www.acmicpc.net/problem/18108)

# 사용 언어
- C++
- C#

# 풀이 방법
- 역시나 별 것 없다..
  문제에서 주어진 불기와 연도의 차이만큼 빼서 출력하면 되겠다.

# 답안 코드

## C++

```cpp
#include <iostream>

int main()
{
	int a;
	std::cin >> a;
	std::cout << a - 543;
}
```

## C#

```cs
using System;

class Program
{
    static void Main(string[] args)
    {
        int a = int.Parse(Console.ReadLine());
        Console.WriteLine(a - 543);
    }
}
```
