# 프로그래밍 기초

## 용어
- 프로그래밍 : 목적을 달성하기 위한 절차를 기술한 것
- 프로그래밍 언어 : 컴퓨터에게 내 생각을 전달하는 언어(도구)
  - 정해진 문법 내에서 작성해야 한다.
- 코드 : 문법에 맞춰 적어놓은 명령어
  - 코딩 : 코드 작성
- 코어 : 문법을 의미
- 라이브러리 : 편하게 프로그래밍 할 수 있도록 도와주는 것

## 컴퓨터의 동작 원리

### 구성 요소
- CPU : 우리가 입력한 명령어의 처리 담당
- 입출력 :
  - Input(입력) : 컴퓨터에 데이터 입력
  - Output(출력) : 컴퓨터의 데이터를 내보내는 행위

#### Bit
- 컴퓨터가 사용하는 데이터 단위. 2가지 상태 표현 가능
  - 0, 1 등..
- Byte 단위로 묶어 여러 상태를 표현한다.
  - 1Byte = 8Bit

### Memory
- 비트 저장공간
- 저장공간은 주소로 표현

### Process
- 프로그램을 실행하기 위한 여러가지 데이터 관리
- 4가지 영역 존재
  - 코드(Code) : 명령어 저장 공간
  - 데이터(Data) : 정적데이터 저장 공간
  - 힙(Heap) : *동적 할당 영역
  - 스택(Stack) : *정적 할당 영역
    - *정적 : 프로그램 실행 전
    - *동적 : 프로그램 실행 중
