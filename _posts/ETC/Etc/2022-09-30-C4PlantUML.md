## 사용하기

### 시작
- `Filename.puml`파일의 맨 위체 `C4_Context.puml` 를 포함해야 한다.
  - 맨 윗 줄에 `!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml` 입력

```
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml

@enduml
```

### 정의 매크로
- `C4_Context.puml`포함 시 C4요소에 정의된 매크로 정의를 사용 가능
    - Person : 사용자
    - Person_Ext : 외부 사용자 (흑백)
    - System : 시스템 구성요소
    - System_Ext : 외부 시스템 구성요소(흑백)
    - Container : 컨테이너 영역
    - Relationship : 관계. 선으로 이을때 쓴다
    - Boundary : 영역을 나눌 때 사용
    - System_Boundary : 시스템 영역을 나눌 때 사용

#### Person, Person_Ext
- Person : 사용자 표시
- Person_Ext : 외부 사용자 표시

```
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml

Person(personAlias, "Label", "Optional Description")
Person_Ext(personAlias, "Label", "Optional Description")

@enduml
```

```PlantUML
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml

Person(personAlias, "User 1", "Description")
Person_Ext(personAlias_Ext, "User 2", "Description")

@enduml
```

#### Boundary, System_Boundary
- Boundary : 영역을 나눌 때 사용
- System_Boundary : 시스템 영역을 나눌 때 사용


```PlantUML
@startuml
!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Container.puml

Person_Ext(personAlias, "Label", "Optional Description")
Container(containerAlias, "Label", "Technology", "Optional Description")
System_Ext(systemAlias, "Label", "Optional Description")

Rel(personAlias, containerAlias, "Label", "Optional Technology")
@enduml
```
