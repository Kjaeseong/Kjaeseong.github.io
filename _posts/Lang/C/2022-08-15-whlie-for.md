# 반복문

## 개요
- while, for 반복문 사용법을 익힌다.

## while
- 조건식이 참일 때 특정 구문을 반복한다.
  - 무한히 반복하므로 break;를 걸거나 조건식을 변경해 중단시킬 필요가 있다.
- 반복문 실행시 조건식을 판단한다
  - 조건식이 false면 한 번도 실행되지 않는다

### 구현예제

```cpp
int num = 1;

while(num <= 5)
{
    // ...
}
```

- 위 예시의 경우 조건식이 참에 해당되므로 무한히 반복한다.
- while문 내부에서 if 분기문으로 break를 실행해도 되고 아래와 같이 바꿔줄 수도 있다.

```cpp
int num = 1;
while(num <= 5)
{
    num++;
    // ...
}
```

## do-while
- while과 같으나 조건식을 실행 후 판단한다.
  - 즉, 반드시 한 번은 실행된다.

### 구현예제

```cpp
int num = 1;

do
{
    // 구문을 먼저 실행 후 조건식을 판별하므로 
    // 1번은 반드시 실행된다.
} while(num == 1)
```

## for
- 반복문 자체에서 변수를 선언해 연산하며 반복한다.
  - 반복 횟수가 정해져 있을 때 편리하다.
- 조건식을 잘못 기입할 경우 마찬가지로 무한반복 될 수 있다.
- 내부 구문에서 if문으로 break;를 걸 경우 도중에 중단시킬 수 있다.

### 구현예제

```cpp
for(int i = 0; i < 5; i++)
{
    // 변수 'i' 초기값 0으로 선언.
    // 'i'가 5 미만이면 실행한다.
    // 구문 실행 후 i에 1을 더한다

    // i가 0 ~ 4일때 실행되므로 총 5회 반복한다.
}
```

## continue
- 반복문 내부에서 남은 구문을 건너 뛰고 다음 반복을 실행한다.

### 구현예제

```cpp
while(1)
{
    //구문 1
    //구문 2
    continue;
    //구문 3

    //구문 1, 2 실행 후 continue로 구문 3을 건너 뛴다.
    //구문 1, 2만 무한 반복된다.
}
```
