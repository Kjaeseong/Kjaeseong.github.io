# var, const, let

| .          | var | const  | let |
|------------|-----|--------|-----|
| 스코프        | 함수  | 블록     | 블록  |
| 초기값 의무할당   | X   | O      | X   |
| 다른 값 할당 가능 | O   | X      | O   |

- JS 사용시 변수에 다른 값을 할당하는 경우는 의외로 적다.
  - 기본적으로 const 선언
  - 다른 값을 할당해야 하는 상황이 생겼을 때 let사용
- Ex>

```javascript
  if(true)
  {
      var V = 3;
      const A = 0;
  }
  console.log(V); 
  // if문 블록 외부에서 접근 가능
  console.log(A);
  // 블록 외부에서 접근 불가능

  const C = 0;
  C = 5;
  // 선언시 초기값이 할당되었으므로 다른 값 할당 불가.

  cosnt C2;
  // 값을 할당하지 않았으므로 에러 발생.

  let L = 0;
  L = 7;
  // 다른 값 할당 가능
```
