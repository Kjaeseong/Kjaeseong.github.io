# 객체 리터럴
- 객체 리터럴에 추가된 편리한 기능들
  - 객체의 메서드에 함수를 연결할 때 콜론(:)과 function을 붙이지 않아도 된다.
  - 속성명과 변수명이 동일한 경우 한 번만 써도 된다.
- 객체 리터럴 안에 동적 속성 선언 가능
- 코드의 중복을 줄일 수 있어 편리하다.

```javascript
var sayNode = function() {
  console.log('Node');
};
var es = 'ES';
var oldObject = {
  sayJS: function() {
    console.log('JS');
  },
  sayNode: sayNode,
};
oldObject[es + 6] = 'Fantastic';
oldObject.sayNode();
oldObject.sayJS();
console.log(oldObject.ES6);
// 기존 리터럴 작성 코드

const newObject = {
  sayJS() {
    console.log('JS');
  },
  sayNode,
  [es + 6]: 'Fantastic',
};
newObject.sayNode();
newObject.sayJS();
console.log(newObejct.ES6);
// 새로운 리터럴 기능 반영 코드
```
