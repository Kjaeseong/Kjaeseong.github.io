# 화살표 함수
- 화살표 함수라는 새로운 함수 추가
  - 기존의 function() {}도 사용 가능
- function 선언 대신 => 기호로 함수 선언.
  - 변수에 대입하면 나중에 재사용 가능
  - 내부에 return문 밖에 없는 경우 return 생략 가능
  
```javascript
function OldFunc(x, y) {
  return x + y;
} 
function OldNot(x) {
  return !x;
}
//기존 함수 사용예시1

const ArrowFunc1 = (x, y) => {
  return x + y;
};
//화살표함수 사용예시1


const ArrowFunc2 = (x, y) => (x + y);
const ArrowNot = x => !x;
//화살표함수 사용예시2
//return문밖에 없는 경우 생략 가능
//반환식을 소괄호로 묶어 연산 가능








```
