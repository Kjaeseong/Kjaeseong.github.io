# async/await
- 노드 7.6부터 지원, ES2017에서 추가
- 노드처럼 비동기 위주 프로그래밍시 도움이 많이 된다.
- 프로미스 코드를 한번 더 깔끔히 줄인다.

```javascript
// 프로미스 코드 예시
function findAndSaveUser(Users) {
  Users.findOne({})
    .then((user) => {
      user.name = 'zero';
      return user.save();
    })
    .then((user) => {
      return Users.findOne({ gender: 'm' });
    })
    .then((user) => {
      // ...
    })
    .catch(err => {
      console.error(err);
    });
}

//async/await문법 적용
async function findAndSaveUser(Users) {
 // try/catch로 로직을 감싸 에러 처리
  try {
    let user = await Users.findOne({});
    // await Users.findOne이 resolve되면 user변수 초기화
    user.name = 'zero';
    user = await user.save();
    user = await Users.findOne({ gender: 'm' });
    // ...
  } catch (error) {
    console.error(error);
  }
};
```

## 화살표 함수에 async 사용

```javascript
const findAndSaveUser = async (Users) => {
  try {
    let user = await Users.findOne({});
    user.name = 'zero';
    user = await user.save();
    user = await Users.findOne({ gender: 'm' });
    // ...
  } catch (error) {
    console.error(error);
  }
};
```

## for문과 async/await
- 노드 10버전부터 지원하는 ES2018문법
- for await of문을 사용해 프로미스 배열 순회
- async 함수의 반환값은 항상 Promise로 감싸진다
  - (= New Promise와 동일하게 동작)
  - 실행 후 then을 붙이거나 또 다른 async함수 안에서 await를 붙여 처리 가능


```javascript
const promise1 = Promise.resolve('성공1');
const promise2 = Promise.resolve('성공2');

// 프로미스 코드 예시
Promise.all([promise1, promise2])
    .then((result) => {
        console.log(result);
    })
    .catch((error) => {
        console.error(error);
    })

// async/await 적용
(async () => {
  for await (promise of [promise1, promise2]) {
    console.log(promise);
  }
})();
```
