# REPL
- 자바스크립트는 스크립트 언어다.
  - 컴파일을 하지 않아도 즉석에서 코드 실행
- 노드에서도 비슷한 콘솔 제공
  - REPL
    - Read : 읽기
    - Eval : 해석
    - Print : 반환
    - Loop : 반복
- 한두 줄의 코드 테스트 용도로 좋다.
- 사용 방법
  - 1. 콘솔 실행
    - Windows : 명령 프롬프트
    - Mac/Linux : 터미널
    - VS Code : 터미널(기본 단축키 : Ctrl + `)
  - 2. node 입력
  - 3. 명령어 입력 
  - 4. 종료방법 
    - Ctrl + C 두 번
    - .exit 입력
  - ex> '>'뒤에 붙은 것은 명령어

```
      > node

      // Read
      > const str = 'Hello world, hello node';
      undefined

      //Read
      > console.log(str);

      //Eval, Print
      Hello world, hello node

      undefined
      // (대기)Loop
```
