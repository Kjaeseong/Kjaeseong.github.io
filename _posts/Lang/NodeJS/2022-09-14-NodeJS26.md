# 기타 모듈들
- 정확한 정보는 [NodeJs 공식 문서](https://nodejs.org/ko/docs/) 참조
  - assert : 값을 비교해 프로그램이 동작테스트시 사용
  - dns : 도메인 이름에 대한 IP주소를 획득시 사용
  - net : HTTP보다 로우 레벨인 TCP/IPC통신시 사용
  - string_decoder : 버퍼 데이터를 문자열로 바꿀 때 사용
  - tls : TLS와 SSL에 관련된 작업시 사용
  - tty : 터미널과 롼련된 작업시 사용
  - dgram : UDP 관련 작업시 사용
  - v8 : V8 엔진에 직접 접근시 사용
  - vm : 가상 머신에 직접 접근시 사용
