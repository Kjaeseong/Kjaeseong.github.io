# 개요
- 2022.08.16 부터 진행하는 개인 프로젝트의 게임 기획안 작성


## 게임 정보
- 게임명 : 미정
- 장르 : 공포 / 퍼즐
- 플레이 인원 : 1인
- 플레이 시점 : 1인칭
- 조작 : 키보드 / 마우스 우선 개발. 추후 VR버전으로 이식 예정

## 플레이 정보
- 플레이 맵은 '매우' 어두운 조명, 손전등을 켜면 불빛이 닿는 범위는 더 잘 보임
- 맵에 몬스터들이 천천히 돌아다니며, 특정 조건에 따라 플레이어 추격
  - 추적 조건(우선순위별)
    - 1. 몬스터의 시야(부채꼴) 내 가장 가까운 플레이어 객체(플레이어 / 마네킹)
    - 2. 구형 범위 내 마네킹 추적
    - 3. 구형 범위 내 '뛰고있는' 플레이어 추적
  - 추적하고 있는 대상이 원형 범위 바깥으로 벗어날 경우 추적을 멈춘다.
- 몬스터와 가까워 질 수록 심박수가 증가
  - 1. 구형 범위 내의 가장 가까운 적들과의 거리(Distance) 계산
  - 2. 가장 가까운 적과의 거리에 따라 4단계로 분할
    - 1 ~ 4단계 순으로 심박수 증가
  - 소리 없이 플레이할 경우를 생각해 UI로도 표현 예정
- 플레이어는 몬스터를 피해 맵 곳곳의 미션을 수행해야 한다.
  - 미션 수행중에도 몬스터들은 이동.
- 미션 및 퍼즐 구상안
  - 미션 : 
    - 일정시간 특정 범위내에서 일정 시간동안 기계 조작
       (기계 객체와 상호작용 후 일정시간 대기 조건)
    - 플레이 맵 내 건축물(폐가)에 숨겨져 있는 비밀번호 쪽지 획득

  - 퍼즐 : 
    - 파이프타일 퍼즐
    - And / Or 논리게이트 퍼즐 
      **퍼즐은 프로젝트 진행 경과에 따라 클리어조건 및 UI 우선구현 후 마지막에 구현 예정
  

## 인게임 플레이
-  필드에 플레이어 소환
   -  시작위치 고정.
- 경광등으로 표시된 탈출지점으로 이동, 미션 수락
- 맵 전역에 흩어져 있는 미션장소에서 미션 수행
- 마지막 미션 수행 완료시 탈출지점 출구 개방
- 탈출구 개방시 120초 내 탈출구 도달시 클리어
   -  게임 내 밸런스에 따라 시간 조절 예정
   -  도달하지 못할 시 Game Over

### 플레이어
- 플레이어 액션
  - 서 있는 상태
  - 걷기 : 속도는 느리나 몬스터의 시야 안으로 들어가기 전까진 추적당하지 않음
  - 뛰기 : 속도가 빠르지만 주변 몬스터에게 추적당함
- 도구
  - 손전등 : 배터리 방전시 On 불가.(최대 게이지 100, 1초에 1씩 감소)
- 체력 : 수치화하지 않고 5회 피격시 플레이어 사망
  - 체력수치별 영상효과 : 
    - 수치가 낮을수록 화면을 흑백으로
- 두려움 수치 : 가장 가까운 몬스터와의 거리에 따라 심박수(소리/UI)증가
  - 두려움 수치별 영상효과 : 1 ~ 4단계 순으로 영상 노이즈 효과
- 인벤토리 :
  - 기본 5칸 화면 하단에 UI 출력
  - 필드에서 아이템 입수시 인벤토리에 보관
  - 플레이어가 원할때 숫자키 1 ~ 5번으로 사용 가능
  - 중간에 있는 아이템 사용시 발생하는 빈칸은 자동으로 정렬

### 몬스터
- 몬스터 액션
  - 평시 : 4초간 전진(걷기) 이동, 랜덤 방향으로 전환 후 반복
  - 뛰기 : 목표 감지시 1초 뜸 들인 후 추적
  - 공격 : 추적중인 플레이어가 일정 거리 내 도달시
    - 쿨타임 : 공격 모션 종료시부터 1.5초
- 게임시작시 선택한 몬스터 수에 따라 필드에 랜덤 배치

### 아이템
- 아이템 종류
  - 마네킹 : 기본체력 7, 플레이어 대신 몬스터의 주의를 끈다
  - 건전지 : 손전등 게이지 40 충전
  - 회복제 : 체력 2 회복
  - 활력제 : 30초간 플레이어 이동속도 3 증가(약 1.5배)
- 획득 방법 : 
  - 최초 스테이지 시작시 각 건축물마다 초기 배치
  - 맵 내 건축물(폐가) 주변에서 120초마다 드랍
- 필드 드랍시 상자 모양으로 표기
- 상자 획득시 모든 아이템 종류 중 랜덤 1개 인벤토리에 추가

### Game Over 조건
- 플레이어 죽음
- 탈출구 개방 후 제한시간 초과

### Stage Clear 조건
- 모든 미션 수행 후 제한시간 내 탈출구 도달

## 구현 희망 목록(구현 완료목록 제외)
- UI
  - 타이틀
  - 인게임
    - 체력바
    - 배터리
    - 인벤토리
    - ESC키 클릭시 계속하기 / 재시작 / 메인화면 선택창
  - Game Over
  - Stage Clear
- 타이틀 씬(게임시작/종료)
- 타이틀 씬 게임시작 선택시 -, + 버튼 마우스 클릭으로 필드 내 몬스터 수 선택(최대 30)
- 탈출구 조건/액션(미션 모두 수행시 문 열림)
- Game Over시 타이틀 복귀 / 재시작 UI
- 마우스 감도 조절 기능
- 적이 가까워질 수록(두려움 수치) 영상 노이즈 효과
- 추후 진행 경과에 따라 수정될 여지 있음
