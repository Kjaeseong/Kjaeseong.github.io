# 개요
- 2022.08.16 부터 진행하는 개인 프로젝트의 게임 기획안 작성

## 게임 정보
- 게임명 : 미정
- 장르 : 공포 / 퍼즐
- 플레이 인원 : 1인
- 플레이 시점 : 1인칭
- 조작 : 키보드 / 마우스 우선 개발. 추후 VR버전으로 이식 예정

## 플레이 정보
- 플레이 맵은 '매우' 어두운 조명, 손전등을 켜면 불빛이 닿는 범위는 더 잘 보임
- 맵에 몬스터들이 천천히 돌아다니며, 플레이어가 적 시야에 들어오면 플레이어와 일정 거리가 벌어질때까지 추적/공격
- 플레이어는 몬스터를 피해 맵 곳곳의 퍼즐을 풀어야 한다.
- 퍼즐을 푸는 동안에도 몬스터들은 이동.
- 몬스터들과 가까워 질 수록 심장박동 소리가 증가해 몬스터가 가까이 있음을 감지
- 퍼즐 구상안
  - 객관식 C++ 문제
  - 열차 레일 잇기 슬라이드퍼즐
  - And / Or 논리게이트 퍼즐
  - 맵에서 단서를 찾아 푸는 숫자 자물쇠(성공시 스테이지 클리어)

### 플레이어
- 플레이어 액션
  - 서 있는 상태
  - 걷기 : 속도는 느리나 몬스터의 시야 안으로 들어가기 전까진 추적당하지 않음
  - 뛰기 : 속도가 빠르지만 주변 몬스터에게 추적당함
- 도구
  - 손전등 : 배터리 방전시 On 불가.(최대 게이지 100, 1초에 1씩 감소)
- 체력 : 수치화하지 않고 5회 피격시 플레이어 사망
  - 체력수치 2 ~ 4 : 낮을수록 화면에 붉은색 페이드
  - 체력수치 1 : 화면 흑백으로 출력
- 인벤토리 :
  - 기본 5칸 화면 하단에 UI 출력
  - 필드에서 '마네킹', '건전지', '회복제' 입수시 인벤토리에 보관
  - 플레이어가 원할때 숫자키 1 ~ 5번 눌러서 쓸 수 있음.
  - 중간에 있는 아이템 사용시 자동으로 빈칸은 자동으로 정렬

### 몬스터
- 몬스터 액션
  - 서 있는 상태(두리번거리기)
  - 걷기 : 두리번거리며 걸어다님
  - 뛰기 : 플레이어 감지시 1초 뜸 들인 후 추적

### 필드 입수 아이템
- 마네킹 : 기본체력 5, 플레이어 대신 몬스터의 주의를 끈다
- 건전지 : 손전등 게이지 충전
- 회복제 : 체력 2 회복

### Game Over 조건
- 플레이어 죽음
- 제한시간 초과

### Stage Clear 조건
- 자물쇠를 풀고 Map 밖으로 이동
