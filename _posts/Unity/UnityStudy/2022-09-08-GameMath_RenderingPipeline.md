# 렌더링 파이프라인
- 세계에 대한 기하학적 표현
- 이 세계를 바라보는 관점을 정의하는 가상 카메라를 이용해 2D 이미지를 만들어내는 역할

## 모델 표현
- 3D모델은 폴리곤(삼각형)으로 이루어져 있다
  - 이 삼각형의 꼭지점을 버텍스라 한다.

### 삼각형
- 3D 물체의 기본 구성요소
  - 물체 구성을 위해 물체 모양과 외형을 묘사하는 삼각형 리스트 필요
- 두르기 순서(winding order) : 삼각형의 버텍스 지정 순서 중요.
- 그림과 수식

### 인덱스
- 모델이 복잡/세밀도가 올라갈수록 중복되는 버텍스의 수 증가
- 인덱스 개념 이용
  - 버텍스 / 인덱스 리스트 구성
    - 버텍스 리스트 : 모든 독특한 버텍스들을 포함
    - 인덱스 리스트 : 삼각형을 구성하기 위한 버텍스리스트의 인덱스 값 포함
  - 인덱스 리스트가 각 버텍스들이 어떻게 두 삼각형을 구성하는지 정의

## 가상 카메라
- 관찰자가 볼 수 있는 세계의 부분 지정
- 공간의 부피는 절두체(frustum)
  - 시야각과 가까운 평면, 먼 평면에 의해 정의된다.
- 투영 윈도우 : 3D장면의 2D포현을 만드는 데 이용될 절두체 내부 3D기하의 2D 영역

## 렌더링 파이프라인
- 단계
  - 로컬 스페이스 - ☆
  - 월드 스페이스 - ☆
  - 뷰 스페이스
  - 후면 추려내기
  - 조명
  - 클리핑
  - 투영 - ☆
  - 뷰포트 - ☆
  - 래스터라이즈
- 파이프라인 내 몇 단계에 의해 하나의 좌표 시스템에서 다른 시스템으로 기하물체를 변환하는 과정이 이뤄진다.

### 로컬 스페이스
- = 모델링 스페이스
- 삼각형 리스트를 정의하는 데 이용하는 좌표 시스템
- 모델링 과정을 쉽고 단순하게 만들어준다.
- 자체의 로컬좌표를 이용하는 것이 월드에서 구현하는 것 보다 쉽다

### 월드 스페이스
- 모델을 전역(월드)좌표로 옮겨 장면을 구성
- 모델은 이동, 회전, 크기변형 등을 변환해 월드스페이스로 옮겨진다.
- 월드 변환은 위치와 크기, 방위를 포함하는 물체 간 관계를 정의하며 이뤄진다

### 뷰 스페이스
- 카메라를 기준으로 각 모델의 좌표들이 다시 계산된다
- 카메라가 월드 내 임의위치/방위를 가지면 투영이나 그 밖의 작업이 어렵거나 덜 효율적

### 투영
- $n$ 차원에서 $n-1$차원을 얻는 과정
- 여러 가지 투영 방법이 있지만 원근 투영에 대해 알아본다.
  - 원근법을 이용해 기하물체 투사
  - 카메라에서 멀리 떨어진 물체는 상대적으로 작게 나타남

### 뷰포트 변환
- 투영 면을 화면의 직사각형으로 변환하는 과정
- 창 크기에 따라 영역이 달라질 수 있다.
